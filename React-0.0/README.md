# React App - Positives and Negatives

A simple React application showcasing the positives and negatives of React framework.

## Features

- **Header**: Displays the main title "Positives and Negatives about React"
- **Main Article**: Contains two sections:
  - ✅ 3 Positive aspects of React
  - ❌ 3 Negative aspects of React
- **Footer**: Shows author name and institution

## Tech Stack

- React 19.2.0
- TypeScript
- Vite
- CSS-in-JS (inline styles)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

1. Clone the repository

```bash
git clone https://github.com/MathiasOliverNielsen/React-0.0.git
cd React-0.0
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint
- `npm run deploy` - Deploy to GitHub Pages

## Deployment

This project is configured for GitHub Pages deployment. To deploy:

```bash
npm run deploy
```
